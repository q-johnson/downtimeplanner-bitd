# Change Log
All major changes to this project will be documented in this file.


## [1.0.0] - 2026-02-15
Initial release of Downtime Planner for Blades in the Dark.

**Added Features**
- **Main Downtime Planner Window**: Plan and manage multiple downtime activities before sending to chat
- **Six Downtime Activities**: Full support for all standard BITD downtime activities:
  - **Acquire Asset**: Track asset acquisition with quality tier, restrictions, and previous acquisition bonuses
  - **Long-Term Project**: Select action rolls and track project progress
  - **Recover**: Choose healing methods (contact or crewmate) with quality/tinker ratings
  - **Reduce Heat**: Document heat reduction methods with appropriate action rolls
  - **Train**: Track training XP for playbook, attributes, and crew upgrades
  - **Indulge Vice**: Manage vice indulgence with purveyor info, stress tracking, and overindulge rules reference
- **Rich Chat Messages**: Formatted chat output with roll reference tables and relevant modifiers
- **Activity Management**: Edit and remove activities before finalizing
- **Data Persistence**: Activities saved to user flags to prevent data loss on window close
- **Trauma Warning**: Alerts players if they have trauma but haven't selected Indulge Vice
- **Overindulge Rules Reference**: Interactive popup showing overindulge consequences
- **Actor Sheet Integration**: Quick access button on character sheets
- **Localization Support**: Full English language support via en.json
